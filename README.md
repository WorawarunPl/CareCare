A full-stack web application designed for mental health care, providing a secure space to log daily moods, notes, and photos, and visualize emotional trends over time.

## ✨ Key Features

* **Daily Journaling:** Securely log your daily mood, detailed notes, and attach a picture to each entry.
* **Mood Flow Analytics:** Visualize and analyze emotional trends over months or years with interactive graphs.
* **Historical Log Viewer:** Browse and review past entries organized by a calendar view or a list, filtered by month and year.
* **Secure Google Authentication:** Seamless and secure email login integration via Google.
* **Enhanced Privacy:** Optional PIN setup to add an extra layer of privacy for sensitive logs.

## 🛠️ Tech Stack & Architecture

* **Front-End:** ReactJS
* **Back-End:** Go (Golang)
* **Database:** PostgreSQL
* **Authentication Service:** Google OAuth 2.0
